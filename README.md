Proyek ini merupakan pengembangan Sistem Manajemen Konten (CMS) berbasis website yang sederhana dan fungsional untuk ARD Organizer Bali. Dikembangkan oleh tim, tujuannya adalah menyediakan website Company Profile yang dapat menampilkan dan mempromosikan layanan event organizer dan paket wisata perusahaan. Sistem ini dirancang agar Admin dapat mengelola informasi dasar website (seperti layanan, galeri, dan kontak) secara mandiri melalui antarmuka web, sehingga tidak diperlukan keahlian pemrograman untuk memperbarui konten. Proyek ini berfokus pada penyediaan platform digital yang terstruktur untuk kebutuhan informasi dan promosi.
